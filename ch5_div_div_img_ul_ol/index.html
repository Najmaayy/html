Learning HTML Structure

This page shows how to use div, ul, ol, and img tags in HTML.

Unordered List (ul)

Use <ul> for bullet points:

  • Apples
  • Oranges
  • Bananas

Ordered List (ol)

Use <ol> for numbered lists:

  1. Wake up
  2. Brush your teeth
  3. Eat breakfast
